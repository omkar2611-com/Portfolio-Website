Predicting Hotel Booking Cancellations

This project aims to predict hotel booking cancellations using machine learning to improve revenue management and operational efficiency in the hospitality sector. By analyzing past booking data, such as lead time, room type, booking source, and previous transactions, a logistic regression model is used to classify bookings as either "cancelled" or "not cancelled." The model identifies patterns leading to cancellations, helping hotels optimize occupancy, streamline operations, and enhance customer satisfaction. It also provides valuable insights for travelers to make better booking decisions, benefiting both customers and the hospitality industry.
